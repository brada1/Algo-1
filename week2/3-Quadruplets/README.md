# Quadruplets

Given four vectors of `N` integer values, compute how many quadruplets (one element from every vector) form a total sum of zero.

```
zero_quadruplets_count(a, b, c, d) {
  ...
  return count
}
```

## Example

```
zero_quadruplets_count([-45, -41, -36, -36, 26, -32],
                       [22, -27, 53, 30, -38, -54],
                       [42, 56, -37, -75, -10, -6],
                       [-16, 30, 77, -46, 62, 45])
=> 5
```
